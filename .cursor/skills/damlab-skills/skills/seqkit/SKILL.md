# seqkit

## Environment

Binary: `bin/seqkit` — relative to this skill directory.

Before issuing any commands, resolve the full absolute path for this machine:
```bash
readlink -f "$(dirname <path-to-this-SKILL.md>)/bin/seqkit"
```
Substitute `<path-to-this-SKILL.md>` with the absolute path you used to read this file.
Use the printed output literally as the first token in every command.
In examples below, `$SEQKIT` is a readable placeholder for that resolved path.

Supports gzip (.gz), xz (.xz), zstd (.zst), and bzip2 (.bz2) input/output natively.

## Subcommands

**Basic Operation**
- `seq` — transform sequences (filter by length/quality, reverse complement, upper/lower case, extract IDs, remove gaps)
- `stats` — simple statistics of FASTA/Q files (read count, length, GC, Q20/Q30, N50); aliases: `stat`
- `faidx` — create FASTA index file and extract subsequences by region (similar to samtools faidx, with regex and full-header support)
- `scat` — real-time recursive concatenation and streaming of FASTX files
- `sliding` — extract subsequences in sliding windows
- `subseq` — get subsequences by region, GTF, or BED file (with up/down-stream flanking)
- `translate` — translate DNA/RNA to protein sequence (all NCBI translation tables, ambiguous bases supported)
- `watch` — monitoring and online histograms of sequence features (for FASTQ)

**Format Conversion**
- `convert` — convert FASTQ quality encoding between Sanger, Solexa, and Illumina
- `fa2fq` — retrieve corresponding FASTQ records by a FASTA file
- `fq2fa` — convert FASTQ to FASTA
- `fx2tab` — convert FASTA/Q to tabular format with optional GC, length, quality columns
- `tab2fx` — convert tabular format (two or three columns) back to FASTA/Q

**Searching**
- `amplicon` — extract amplicon (or flanking region) via primer pair; supports mismatches and degenerate bases
- `fish` — look for short sequences in larger sequences using local alignment
- `grep` — search sequences by ID, name, sequence motif, or regex; mismatch allowed
- `locate` — locate subsequences/motifs with mismatch support; outputs BED or GTF

**Set Operations**
- `common` — find common/shared sequences across multiple files by ID, name, or sequence
- `duplicate` — duplicate sequences N times; aliases: `dup`
- `head` — print the first N records (or records until total length >= L)
- `head-genome` — print sequences of the first genome sharing a common prefix in name
- `pair` — match up paired-end reads from two FASTQ files
- `range` — print FASTA/Q records in a start:end range
- `rmdup` — remove duplicated sequences by ID, name, or sequence
- `sample` — sample sequences by number or proportion (loads all into memory with `-n`)
- `sample2` — sample sequences v2 — unbiased fixed-size sampling, memory efficient
- `split` — split sequences into files by ID, region, part size, or number of parts
- `split2` — split sequences by part size or count; supports paired-end FASTQ

**Edit**
- `concat` — concatenate sequences with the same ID from multiple files
- `mutate` — edit sequence (point mutation, insertion, deletion)
- `rename` — rename duplicated IDs by appending `_N` suffix
- `replace` — replace name/sequence by regular expression; supports `{nr}`, `{kv}`, `{fn}`
- `restart` — reset start position (rotate) for circular genomes; aliases: `rotate`
- `sana` — sanitize broken single-line FASTQ files

**Ordering**
- `shuffle` — shuffle sequences (in-memory or 2-pass for FASTA)
- `sort` — sort sequences by ID, name, sequence, or length (in-memory or 2-pass for FASTA)

**BAM Processing**
- `bam` — monitoring and online histograms of BAM record features

**Miscellaneous**
- `merge-slides` — merge sliding windows generated by `seqkit sliding` into BED3 format
- `sum` — compute message digest (seqhash) for all sequences in FASTA/Q files

## Common patterns

**File statistics (read count, length distribution, GC content):**
```bash
$SEQKIT stats -a input.fq.gz
# -a includes N50, Q20/Q30% for FASTQ; use for multiple files:
$SEQKIT stats -a *.fq.gz
```

**Filter reads by length:**
```bash
# Keep reads >= 1000 bp
$SEQKIT seq -m 1000 input.fq.gz -o filtered.fq.gz
# Keep reads between 500 and 5000 bp
$SEQKIT seq -m 500 -M 5000 input.fq.gz -o filtered.fq.gz
```

**Convert FASTQ to FASTA:**
```bash
$SEQKIT fq2fa input.fq.gz -o output.fa.gz
```

**Grep sequences by ID list:**
```bash
# ids.txt: one ID per line
$SEQKIT grep -f ids.txt input.fa.gz -o subset.fa.gz
# By pattern (regex):
$SEQKIT grep -r -p "^contig_[0-9]+" input.fa -o contigs.fa
```

**Random subsample:**
```bash
# By count:
$SEQKIT sample -n 10000 -s 42 input.fq.gz -o sampled.fq.gz
# By proportion:
$SEQKIT sample -p 0.1 -s 42 input.fq.gz -o sampled.fq.gz
```

**Extract subsequences by region:**
```bash
# BED-based:
$SEQKIT subseq --bed regions.bed genome.fa -o regions.fa
# By coordinate (1-based):
$SEQKIT subseq -r 100:200 input.fa
```

**Reverse complement:**
```bash
$SEQKIT seq -r -p input.fa -o revcomp.fa
# -r: reverse complement; -p alone: complement only; -r alone: reverse only
```

**Split paired-end FASTQ into per-file chunks:**
```bash
$SEQKIT split2 -1 R1.fq.gz -2 R2.fq.gz -s 1000000 -O split_out/
```

**Remove duplicate sequences:**
```bash
$SEQKIT rmdup -s input.fa -o deduped.fa
# -s: by sequence; default is by ID
```

## Piping

seqkit reads/writes stdin/stdout cleanly:
```bash
$SEQKIT seq -m 1000 input.fq.gz | $SEQKIT sample -n 5000 -s 1 > sampled_long.fq
```
